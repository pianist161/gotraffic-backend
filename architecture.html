Core Algorithm — ForceOff → SEPAC Split Conversion
Dual-Ring 8-Phase Structure
Ring A
Ph1(left) → Ph2(thru) ‖ Ph3(left) → Ph4(thru)
Ring B
Ph5(left) → Ph6(thru) ‖ Ph7(left) → Ph8(thru)
Split Formulas
Ph1 = FO₁ − FO₄
Ph2 = CL − FO₁
Ph3 = FO₃
Ph4 = FO₄ − FO₃
Ph5 = FO₅ − FO₈
Ph6 = CL − FO₅
Ph7 = FO₇
Ph8 = FO₈ − FO₇
